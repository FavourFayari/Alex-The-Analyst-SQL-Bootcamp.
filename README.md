## SQL Data Analytics Bootcamp

This repository contains my daily practice scripts as I master SQL for data analytics. Inside, I am building relational databases from scratch, handling data types, and running multi-table analysis.

## Skills Demonstrated
* **Database Design:** Creating tables using explicit data types (`INT`, `VARCHAR`, `DATE`).
* **Data Manipulation:** Populating records using `INSERT INTO`.
* **Advanced Querying:** Joining isolated tables via `INNER JOIN` and restructuring outputs with custom table aliases.
* **Conditional Logic:** Building data-driven categories using `CASE WHEN` statements.

## Featured File
* `bootcamp_practice.sql`: My master tracking script containing all schemas, mock data (featuring *The Office* dataset), and analytical queries.
